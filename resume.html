I3IPC-Python

The ‘i3ipc-python’ is an Inter-Process Communication (IPC) library for interacting with the tiling window managers ‘i3wm’ and ‘Sway WM’ for the GNU/Linux desktop.

I have two accepted Pull Requests to the main project repo currently:

  1. https://github.com/altdesktop/i3ipc-python/issues/170: The ‘SwayWM’ version of the library was missing an API that returns all GUI windows belonging to a given app by name.
  2. https://github.com/altdesktop/i3ipc-python/issues/172: The same framework was also missing an API that was part of the SwayWM’s IPC specifications, so I implemented it.
